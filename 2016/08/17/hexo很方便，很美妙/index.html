hexo很方便，很美妙

发表于 2016-08-17   |  

重装系统后，真是不想再去折腾一遍octopress了，实在需要太多配置。osX EICapion 下升级个ruby都是整死人。果断改用hexo，5分钟就能搞定。再把octopress的md文件直接复制过来，就行。

安装

官方网站：https://hexo.io/

1.安装node.js，官方就推荐了安装方式。
The best way to install Node.js is with nvm.

2.安装命令一行搞定

$ npm install -g hexo-cli

配置

  1. 主题选用nexT，非常简洁，中文文档也很细致。

  2. 记住开启分类或者标签都需要先执行$ hexo new page tags，并且在source中修改md文件，增加type: "tags"，文章中用-这种标记来写分类和标识。直接在冒号后写，会看不见，不知道为什么。

    categories:
- Sports
- Baseball
tags:
- Injury
- Fight
- Shocking

  3. 分类的名称和文件夹名称是可以独立配置的，如

    default_category: uncategorized
category_map:
编程: programming
生活: life
其他: other

开始吧

不追求过多细节，就这么开始写吧。

$ hexo new [layout] <title>
$ hexo server

参考：http://div.io/topic/1691 里面很详细，我也是照着做的。
