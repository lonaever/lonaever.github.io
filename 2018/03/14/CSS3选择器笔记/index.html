CSS3选择器笔记

发表于 2018-03-14   |   分类于 CSS   |  

属性选择器

基本语法

[xxx=xxx]{}

CSS3中新增3个具有模糊匹配的写法

  • [xxx*=xxx]{} 相当于contain
  • [xxx^=xxx]{} 相当于startWith
  • [xxx$=xxx]{} 相当于endWith

伪元素选择器

诸如：

  • p:first-line{}
  • p:first-letter{}
  • p:before{} 在p前面加入一些字符
  • p:after{ content: "--"}

伪类选择器

  • :root{} 表示整个根页面。如果设置了这个属性，再设置body就只表示整个内容部分了。
  • :empty{} 可以选择到所有空内容的标签，比如<td></td>
  • div *:not(h1){} 表示div下的不是h1的标签
